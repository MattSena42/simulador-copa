# 🏆 Simulador da Copa do Mundo

> Aplicação web desenvolvida para calcular e simular a pontuação de uma seleção na fase de grupos da Copa do Mundo.
> O projeto resolve um desafio de lógica de programação, utilizando manipulação de estado, programação funcional e uma interface moderna em Dark Mode.

---

## 📄 Demonstração

- 🌐 **Deploy no GitHub Pages:** Acesse a aplicação rodando ao vivo diretamente no seu navegador: [https://mattsena42.github.io/simulador-copa/](https://mattsena42.github.io/simulador-copa/)

---

## 📌 Funcionalidades

- ✅ **Cálculo Dinâmico:** Avaliação instantânea de Vitória (3 pts), Empate (1 pt) ou Derrota (0 pts) com base no placar.
- ✅ **Histórico de Partidas:** Exibição detalhada de saldo de gols e situação atual (ex: "Excelente campanha", "Ainda pode classificar") após cada jogo inserido.
- ✅ **Estatísticas Avançadas:** Ao registrar o limite de 3 jogos da fase de grupos, um painel final exibe o total de pontos, gols marcados/sofridos, saldo, número de vitórias/empates/derrotas e % de aproveitamento.
- ✅ **Classificação Automática:** Aplicação da regra simplificada para ditar o destino da seleção (Classificada com excelente campanha, Depende de critérios ou Eliminada).
- ✅ **Design Responsivo:** Interface limpa, moderna e adaptável a telas menores utilizando utilitários CSS.

---

## 🚀 Tecnologias utilizadas

- **HTML5** (Estrutura semântica)
- **JavaScript ES6+** (Lógica baseada em Arrays e métodos como `.reduce()` e `.filter()`)
- **Tailwind CSS** (Estilização via CDN)

## 💻 Como usar

1. Clone o repositório:
    ```bash
    git clone [https://github.com/MattSena42/simulador-copa.git](https://github.com/MattSena42/simulador-copa.git)
    ```

2. Acesse a pasta do projeto:
    ```bash
    cd simulador-copa
    ```

3. Como o projeto não possui dependências de backend, basta abrir o arquivo `index.html` em qualquer navegador moderno para executar a aplicação localmente. Alternativamente, você pode usar a extensão *Live Server* do VS Code.
